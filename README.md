# MW-Sense-Inventory


**data** folder contains: 
1. Sample aligned version of the MW.
2. Sample (5 lemmas) annotated version of the MW. 
3. The sample original clusters of the aligned MW.

**embeddings** folder contains: <br/>
Sample aligned version's sense embeddings generated using the 5 different models.

**model-method-clusters** folder contains: <br/>
The cluster outputs of all model-method pairs.

**results** folder contains: <br/>
A comparison of the count of clusters generated by the model-method pairs for each lemma in the sample aligned version.
